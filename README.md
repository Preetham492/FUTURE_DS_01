Ecommerce_Sales_Dashboard

📝 Problem Statement

Ecommerce businesses generate large volumes of transactional data, making it challenging to analyze performance and extract meaningful insights.
The key challenge is to monitor overall sales, profitability, customer behavior, and regional performance in an efficient and visual way.

This project aims to build an interactive Power BI dashboard that provides a clear overview of ecommerce sales performance using KPIs and visual analytics.
The dashboard helps stakeholders identify trends, top-performing categories, customer preferences, and areas for improvement to support better decision-making.

🎯 Project Objectives
To analyze Total Sales and Total Profit
To track monthly profit trends and performance
To identify top-performing product categories and sub-categories
To analyze sales distribution across different states
To understand customer purchasing behavior
To evaluate payment mode preferences
To monitor overall business performance
📌 Key KPIs
Total Sales Amount: 438K
Total Profit: 37K
Total Quantity Sold: 5615
Average Order Value (AOV): 121K
📈 Dashboard Features
Visuals Included:
KPIs Cards → Total Sales, Profit, Quantity, AOV
Sales by State (Bar Chart) → Regional performance comparison
Quantity by Category (Donut Chart) → Category contribution
Profit by Month (Bar Chart) → Monthly profit trends
Sales by Customer (Bar Chart) → Top customers
Quantity by Payment Mode (Donut Chart) → Payment preferences
Profit by Sub-Category (Bar Chart) → Product-level profitability
