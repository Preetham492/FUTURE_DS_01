## 📝 Problem Statement  
Ecommerce businesses generate large volumes of transactional data, making it difficult to extract meaningful insights efficiently.  
The main challenge is to track overall sales performance, profitability, customer behavior, and regional contribution in a structured and visual manner.  

This project aims to build an **interactive Power BI dashboard** that provides a clear and detailed view of ecommerce performance using KPIs and visual reports.  
The dashboard helps stakeholders make better decisions by identifying high-performing categories, customers, payment methods, and regions.  

---

## 🎯 Project Objectives  
- To analyze **Total Sales** and **Total Profit**  
- To track **Monthly Profit Trends**  
- To identify **Top Customers and Sub-Categories**  
- To analyze sales performance by **State**  
- To compare **Category-wise Sales Contribution**  
- To understand sales distribution by **Payment Method**  
- To monitor overall business performance

## 📈 Dashboard Features  
### Visuals Included:  
- KPI Cards (Sales, Profit, Quantity, AOV)

- Quantity by Category (Donut Chart)  
- Profit by Month (Bar Chart)  
- Sales by Customer (Bar Chart)  
- Quantity by Payment Mode (Donut Chart)
